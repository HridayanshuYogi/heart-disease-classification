# Heart Disease Classification
Project Overview

This project aims to predict whether a person has heart disease based on medical attributes using classification algorithms. It is a binary classification problem where the output is either presence (1) or absence (0) of heart disease.

# Dataset
Source: Kaggle (Heart Disease Dataset)
Features include:
Age
Sex
Chest pain type
Resting blood pressure
Cholesterol levels
Maximum heart rate
Other medical indicators

# Steps Performed
1. Data Collection
Dataset collected from Kaggle
2. Data Preprocessing
Checked for missing values
Feature scaling using StandardScaler
Train-test split
3. Data Visualization
Correlation heatmap
Target variable distribution
Feature relationships
4. Model Training
Applied classification algorithms:
Logistic Regression
Random Forest Classifier
5. Model Evaluation
Evaluated using:
Accuracy Score
Confusion Matrix
Classification Report (Precision, Recall, F1-score)
6. Hyperparameter Tuning
Used GridSearchCV to find best parameters
Observed impact of parameters on model performance
# Technologies Used
Python
Pandas, NumPy
Matplotlib, Seaborn
Scikit-learn
